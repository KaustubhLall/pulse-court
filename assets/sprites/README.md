# Pulse Court sprite pack

This is an original, transparent PNG sprite pack for the Pulse Court viewer.
It is deliberately separate from the deterministic core, so it has no effect
on headless simulation, replay hashes, or training throughput.

## Sheets

`animated/` contains the alpha-converted source sheets used by the viewer:

- Character sheets use their filename grid, such as `*_idle_4x4.png`,
  `*_strike_6x4.png`, and `*_dash_5x4.png`.
- Effect sheets use their filename grid, such as `anchor_well_fx_6x3.png` and
  `goal_burst_4x3.png`.
- `ability_icons_3x1.png` is ordered Kite, Vale, Bastion.

Use the normalized grid metadata in `animated/manifest.json` rather than
hard-coded pixel dimensions.

`assets/sprites/clean/` contains runtime-cleaned player body sheets
(generated by `scripts/clean_sprites.py`). The viewer samples these clean sheets
for all player body states; the raw `animated/` player body sheets remain source
art.

All sheets have transparent alpha backgrounds and were created as original
project art. They contain no third-party game characters, logos, or UI.
